# ViralScope Privacy Policy

**Last updated: May 2026**

## Overview
ViralScope is a personal, non-commercial web dashboard that reads 
publicly available Reddit post data for personal trend analysis.

## Data Accessed
- Post titles, scores, upvote ratios, comment counts, timestamps,
  subreddit names, and permalinks
- All data is publicly visible on Reddit
- No private messages, user accounts, or restricted content is accessed

## Data Storage
- Data is stored temporarily in local memory only
- No data is retained beyond 48 hours, in compliance with 
  Reddit's Data API Terms
- No database, cloud storage, or third-party service is used

## Data Usage
- Data is used solely for personal, non-commercial trend analysis
- Data will never be sold, licensed, shared, or redistributed
- Data will never be used to train any AI or machine learning model

## Privacy
- No Reddit user data or personal information is collected
- No attempt is made to identify, track, or profile any Reddit user
- No sensitive user characteristics are inferred
